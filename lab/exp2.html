The following web page describes about the Tags and Semantics in HTML5 and we will learn the same about a few tags.

All about Tags in HTML5

Commonly used Tags:
<a>: This defines a Hyperlink
<abbr>: This defines an abbreviation or an acronym
<br>: This defines a single line break
<img>: This defines an image in the tag
<li>: This defines a list item
New tags in HTML5:
<article>: This defines an article
<canvas>: This is used to draw graphics, on the fly, via scripting (usually JavaScript)
<keygen>: This defines a key-pair generator field (for forms)
<menuitem>: This defines a command/menu item that the user can invoke from a popup menu
